# WDD 330 - Web Frontend Development II
## Final Project - Find Your Anime

### Description

Find your anime is a web application designed to help users find anime suggestions based on a query or personal preferences. It saves users time by helping them discover the “perfect” anime to watch.


### Common Workflow Commands

- `npm run lint` to run ESLint against your code to find errors.
- `npm run format` to run Prettier to automatically format your code.
- `npm run start` starts up a local server and updates on any JS or CSS/SCSS 
- `npm run build` to build final files when you are ready to turn in.
